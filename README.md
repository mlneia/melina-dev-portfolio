# Melina Bazrafkan | Developer Portfolio

A responsive personal portfolio introducing Melina&apos;s background, current technical skills, projects, and BCIT learning path.

## Built with

- Semantic HTML5
- Modern CSS
- Flexbox and CSS Grid
- Google Fonts: Manrope and Fraunces
- Responsive media queries for tablet and mobile layouts
- CSS-only light and dark theme control
- No JavaScript or external runtime dependencies

## Structure

```text
melina-dev-portfolio/
├── index.html
├── README.md
├── css/
│   └── styles.css
└── docs/
    └── assignment-notes.md
```

Open `index.html` in a browser to view the site. The layout adapts for desktop, tablet, and mobile screens.
