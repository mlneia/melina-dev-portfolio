## Project purpose

This project is a personal developer portfolio that introduces who I am, what I am learning, and the technical skills I am building. It is designed as a professional, expandable foundation rather than a one-time assignment page.

## COMP2132 Assignment 01

Version 1 focuses on semantic HTML, responsive CSS layouts, accessible navigation, external JavaScript, and basic DOM manipulation. The page includes personal introduction content, technical skills, transferable experience, education focus areas, projects, and professional contact links.
